## WHAT EACH PART DOES

### 🖥️ Tracking Script
- Injected into your website (via Landingsite.ai)
- Tracks: page views, clicks, form interactions, scrolls, time
- Runs invisibly in background
- Sends events to your backend API

### 📡 Backend API
- Receives tracking events from website
- Stores them in Supabase database
- Provides analytics endpoints for dashboard
- Deployed on Vercel (free tier)

### 📊 Dashboard
- Admin panel to view analytics
- Real-time metrics display
- Funnel analysis (where users drop off)
- User session tracking
- Deployed on Vercel (free tier)

### 💾 Database
- Stores all tracking events
- On Supabase (free tier)
- Automatically backs up daily
- Secured with authentication
